# Модули

Модули CapsuleUI — это переиспользуемые функциональные пакеты, которые расширяют возможности вашего приложения. В отличие от компонентов, модули предоставляют логику и утилиты, которые можно использовать в нескольких компонентах.

## Доступные модули

### Form

- **[Модуль Form](/ru/modules/form)** - Полная система валидации форм с управлением полями и обработкой ошибок

## Добавление модуля

Чтобы добавить модуль в ваш проект, используйте:

```bash
npx capsule module add <module-name>
```

Например, чтобы добавить модуль Form:

```bash
npx capsule module add form
```

См. документацию [Команды CLI](/ru/cli-commands) для получения дополнительной информации об управлении модулями.

## Модуль vs Компонент

- **Компоненты** — это UI-элементы, которые можно видеть и с которыми можно взаимодействовать (кнопки, поля ввода и т.д.)
- **Модули** — предоставляют функциональность и логику, которые улучшают ваше приложение (валидация, утилиты и т.д.)

Модули добавляются в `@capsule/modules` и автоматически импортируются в главный файл `@capsule/index.js`.
